## ■ **Unity_3DRPG**
### ○ 개발기간 : 2024-11-13 ~ 2024-11-14
### ○ 개발도구 : Unity, C#
### ○ 스파르타코딩 - 유니티 심회 주차 개인과제
### ○ 3D 게임
### ○ 키 설명
- WASD : 이동
- 스페이스 : 점프
- 마우스 오른쪽 버튼 : 공격(콤보)
- 탭 : 인벤토리
### ○ 구현기능
- 기본 UI 구현
- 아이템 및 장비 창 UI 구현
- 스테이지 시스템(스테이지 클리어시 다음 스테이지 시작)
- ScriptableObject를 이용한 데이터 관리
- 랜덤 맵 생성 기능(미로)
